## Dataset

Dataset name: **Pass-Fail Data** 

This dataset contains student performance-related features such as attendance, homework percentage, midterm score, and study hours per week.

## Machine Learning 
Problem Type: Classification (Binary Classification)

Goal: Predict whether a student will **pass** (1) or **fail** (0) using the input features:

**attendance_pct, homework_pct, midterm_score, study_hours_per_week**.

Why Classification? Because the target variable pass has two classes: 0 and 1
